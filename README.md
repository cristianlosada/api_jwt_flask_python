## ejercicio practico del uso de jwt con python y flask, microframework web..

- Instale su entorno
- Habilite el archivo que guarda las llaves para montar el api  
- Instale dependecias del archivo requirements.txt


al preparar el entorno 

se inicia el servidor con 'py main.py'

y se ejecutan las pruebas con postman al servidor de localhost a la ruta /api/login para obtener el token y /api/verify/token para obtener la data que guarda el token.
